🫀 Heart Disease Data Analysis (UCI Dataset)
📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of the world-famous UCI Heart Disease Dataset (Cleveland). The goal is to identify how clinical parameters like Cholesterol levels, Age, and Blood Pressure correlate with the risk of heart disease in patients.

As a Pharmacy student, I have used this real-world dataset to bridge the gap between clinical knowledge and data-driven insights.

📊 Key Insights & Visualizations
Cholestrol_level_vs_Heart_Disease_Risk.png
In this analysis, I used Violin Plots to visualize the distribution of cholesterol across 'Healthy' and 'High Risk' groups.

Observation: The density of cholesterol levels is higher in the 240-280 mg/dl range for the 'High Risk' group compared to the 'Healthy' group.

Statistical Distribution: The violin plot reveals not just the median, but the entire probability density of the data, including outliers in the healthy population.

🛠️ Tech Stack
Language: Python

Libraries: * Pandas (Data Loading & Cleaning)

Seaborn & Matplotlib (Advanced Data Visualization)

NumPy (Numerical Operations)

Dataset Source: UCI Machine Learning Repository

🧪 Methodology
Data Acquisition: Fetching raw data directly from the UCI repository via URL.

Data Cleaning: Handling missing values (NaN) and ensuring correct data types for clinical attributes.

Feature Engineering: Mapping target variables (0 to 4) into categorical 'Healthy' and 'High Risk' classes for better interpretability.

Statistical Visualization: Implementing Violin plots to understand data spread and density.

💡 Pharmaceutical Relevance
This project demonstrates how a Pharmacist can utilize Data Science to:

Identify high-risk patient profiles early.

Understand the correlation between biomarkers (like Cholesterol) and chronic diseases.

Support clinical decision-making with visual evidence.
